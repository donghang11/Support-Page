After clicking the plus mark beside File will go to add file page:

Please do select a sender in Detail Action page

The adding a new file page:


Below the sitemap are:
  • Name: the name of the file
  • CASE: The description of the related case, cannot be changed.
  • Path: the path where the file is located
  • Sender: The member who attaches this file, inherited from solution, cannot be changed
  • Note: The note for this file
  • Date: The date when this file is attached, cannot be changed
  • Delete: If a user wants to delete this file, just click this button and confirm the pop up alert.
After fill in the information of each field, the information will be automatically saved into database.
